# TaskFlow-GerenciadorDeTarefas
MVP acadêmico de gerenciador de tarefas com autenticação de usuários. CRUD completo, senhas criptografadas (bcrypt), sessão (express-session), interface responsiva (Bootstrap 5) e código 100% comentado. Stack: Node.js, Express, Sequelize, SQLite, Handlebars.
